                              1hella Neil a year ago

                              I've done it here: https://a1hella-voting.herokua...
                              I'm using AngularJS and the library angular-chart.js: http://jtblin.github.io/angula...

                              You can set up the html like this: <canvas class="chart chart-doughnut" data="myData" labels="myLabels"></canvas>

                              The chart won't render if there's nothing in "myData" or "myLabels"; what worked for me was initializing them as dummy data first before getting the real data so I did this in the controller: "$scope.myData = [1, 1]; $scope.myLabels = ["label1", "label2"]".

                              Then once you get the data from JSON or wherever just replace myLabels and myData with whatever data you need.

                              The repository for that site is here: https://github.com/1hella/free...

                              The relevant files are client/app/main/main.html and main.controller.js, and client.app/poll/poll.html and poll.controller.js.
